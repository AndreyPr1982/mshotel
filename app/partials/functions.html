Основные функции

  • наглядный график бронирований и заселений в виде шахматки
  • наглядный график уборки с разделением по горничным
  • наглядный график предоставления услуг по часам (баня, сауна, инвентарь и т.п.)
  • возможность гибкого ценообразования
  • печать всех необходимых документов (анкета, счет, договор, акт и др.)
  • база клиентов (ФЛ, ЮЛ) с возможностью поиска и просмотра истории
  • возможность автоматического сканирования паспорта клиента из программы
  • определение загрузки номерного фонда (%) на любую дату
  • различные стратегии расчета стоимости (с учетом Ваших требований)
  • учет разницы между стоимостью по документам и реальной стоимостью проживания
  • формирование различных отчетных ведомостей
  • возможность одновременной работы в программе по сети (в т.ч. удаленно)
  • разделение прав доступа с помощью учетных записей пользователей
  • возможность выписывать/аннулировать карты для управления электронными замками
